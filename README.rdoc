== README

Open To-do API

Build an externally useable API for a basic to-do list application.

User Stories

* As the Open To-do API, I want to return JSON representations of users, lists and items

* As a user, I want to authenticate myself using an email and password

* As a user, I want to create new users, lists and items

* As a user, I want to remove users and lists

* As a user, I want to update list and item attributes
